Data Entry Form

Version Date Features
Version 1.1 16th Dec, 2019
  • Carry forward Cemetery Name, Section, and Marker Type.
  • Autogenerate Primary Keys.
  • Masks on all dates fields.
Version 1.0 9th Dec, 2019
  • Header, buttons, and images are fixed.
  • The Open Image button will open a resizable window of the current record, which remains is sync with the records displayed on the main form.
  • The Emblem drop-down menu show a small picture of the emblem.
  • The Sequence ID and Primary Key fields are not editable, but the text can be selected (to copy and paste).
  • Gravesite number is mandatory for every record.
  • For each decedent (Primary, Secondary, etc.), if any of the fields are filled, then the Last Name field becomes a mandatory field.
  • Short-cut keys to enter "ILLEGIBLE" or "UNKNOWN" in text-boxes.
